# ShoppyGlobe Backend API

## Project Description
ShoppyGlobe Backend API is a RESTful backend application built using Node.js, Express.js, and MongoDB.  
The project provides APIs for user authentication, product management, and cart operations using JWT authentication.

---

## GitHub Repository
https://github.com/Saad-Shams-dev/shoppyglobe-backend

---

## Technologies Used
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcryptjs
- Thunder Client

---

## Features Implemented

### User Authentication
- User Registration API
- User Login API
- JWT Token Generation
- Protected Routes using Middleware

### Product APIs
- Get All Products
- Get Single Product by ID

### Cart APIs
- Add Product to Cart
- Update Cart Quantity
- Delete Cart Item

---

## API Testing
All API routes were tested successfully using Thunder Client.

## MongoDB Collections
The following collections were created and tested in MongoDB Atlas:
- users
